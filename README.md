# Node.js & Prisma CRUD Tutorial

A beginner-friendly tutorial created in collaboration with the Scientific Club at my university to teach students how to build a RESTful API with Node.js, Express and Prisma.

The API demonstrates CRUD operations, introducing students to:

- Using Prisma ORM for database interaction  
- Structuring models and routes in Node.js  
- Handling Create, Read, Update, Delete operations efficiently  


## Tech Stack

- Node.js  
- Express.js  
- Prisma ORM  
- SQL Database (MySQL or PostgreSQL)
