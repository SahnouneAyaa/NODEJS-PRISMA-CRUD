# 🧩 Node.js & Prisma CRUD Tutorial

A beginner-friendly tutorial created in collaboration with the Scientific Club at my university to teach students how to build a RESTful API using Node.js, Express, and Prisma.

---

## Overview
This project is part of a collaborative workshop series where I contributed as an instructor, helping design and deliver sessions focused on teaching students how to build backend APIs.

Each session of the workshop covered a different backend technology (MongoDB, GraphQL, Prisma, Sequelize, SQL), and this project represents the **Node.js + Prisma module**.

The goal was to introduce students to real-world backend development using Prisma ORM with relational databases.

---

## Features

This API provides basic CRUD operations for a user and house management system:

### User Management
- Create user
- Get all users
- Update user
- Delete user

### House Management
- Create house for a user
- Get all houses
- Get house by ID
- Get houses for a specific user

---

## Prisma & Database Concepts
This project introduces key Prisma and relational database concepts:

- Using Prisma ORM for database operations
- Defining and managing database models
- Understanding **one-to-many relationships** (User → Houses)
- Structuring relational queries efficiently
- Performing CRUD operations with Prisma Client

---

## Tech Stack

- Node.js  
- Express.js  
- Prisma ORM  
- SQL Database (MySQL or PostgreSQL)
